## 🛠주요 기능

1. React를 이용한 SPA 웹 사이트 제작
2. ESPN API를 이용하여 축구 통계데이터 제공
3. 라이트모드/다크모드 변경 기능 추가(v1.2.0)

## 🚧기능 개선

1. 기존 UI shadcn/ui로 교체 (v1.1.0)
2. 공통 함수 모듈화 (v1.2.0)
3. 깔끔하고 직관적인 인터페이스 제공을 위해 shadcn/ui -> daisyui로 교체 (v1.2.0)
4. 로고 교체 (v1.2.0)
